# 📋 Kanban — Pracovní Průvodce

Tento Kanban sleduje práci na projektech a plání.

## Sloupce

| Sloupec | Popis |
|---------|-------|
| **Backlog** | Nové nápady a plány |
| **In Progress** | Zrovna řeším (dnes) |
| **Review** | Čeká na kontrolu |

1. **Backlog** — všechny nové nápady
2. **In Progress** — právě teď řeším
3. **Review** — připraveno pro revizi
4. **Done** — hotovo dnes

## Daily Updates

Každý den upravte `in-progress.md` a `done.md` podle Nightly Build výsledků.
