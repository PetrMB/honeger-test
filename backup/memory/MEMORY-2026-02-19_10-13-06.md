#### Sync reminders výsledek

**Format notes v reminderu:**
```

🏪 Lidl
📅 čt 19. 2. – ne 22. 2.
```

#### Postup
1. `remindctl list Nákupy --json` → JSON 20 reminderů
2. `shopping-deals.py --json` → uloží do `data/shopping-deals.json`
3. `update-reminders-with-prices.py` → najde matchy, upraví notes

#### Skript: `update-reminders-with-prices.py`
- `normalize_title()` odstraní [342] číslování, 2x/3x množství
- `find_deal()` → full match → contains match → partial match (subword)
- `update_reminder_notes(reminder_id, price_info)` → `remindctl edit <id> --notes "<text>"`

#### Úpravy JSON formatu (shopping-deals.json)
```json

  "count": 6,

      "amount": "6 l",

    },
    ...
  ]
}
```
