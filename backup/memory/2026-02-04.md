### Jak to funguje:
- Skill automaticky **mluví** když zpráva začíná: `User talked via voice recognition`
- Používá macOS `say` příkaz (lokální TTS, bez cloudu)
- Hlas: **Zuzana** (český ženský hlas)

### Použití:
```bash
# Základní test
printf 'Ahoj Petře!' | say -v Zuzana

# Rychlejší řeč
printf 'Text' | say -v Zuzana -r 220

# Pomalejší řeč  
printf 'Text' | say -v Zuzana -r 160
```

### Voice Wake integrace:
- Když mluvíš přes Voice Wake → Otto automaticky odpoví **hlasem**
- První acknowledge (potvrdí že slyšel)
- Pak provede úkol
- Nakonec může znovu mluvit s výsledkem
