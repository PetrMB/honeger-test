#### Diagnóza

- Pages API **nepodporuje Direct Upload** přes REST API (dokumentace potvrzuje: "no documented REST API for direct file uploads")
- Wrangler CLI chce OAuth login (není automaticizovatelný pro skripty)
- GitHub webhook nefunguje (nebyl triggered)

#### Závěr

**Cloudflare Pages API je omezené** - nelze automatizovat deploy přes API bez OAuth.

### 🎯 Návrh dalšího postupu

Pro konkrétní projekt (test1, test2, ...) pokračovat ručně:
1. Vytvořit Pages projekt v Cloudflare dashboardu (připojit GitHub)
2. Nastavit custom domain přes UI (protože API má bug s TLD)
3. Po provedení ručního kroku je vše automatické přes GitHub commits
